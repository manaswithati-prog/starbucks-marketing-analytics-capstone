# Starbucks Marketing Analytics & Brand Strategy
### End-to-End Capstone Project | Manaswi Thati | WSU MS Marketing Analytics & Insights

## Project Overview
An 8-week end-to-end marketing analytics capstone analyzing Starbucks customer behavior, competitive positioning, digital channel performance, and 90-day brand strategy — built to demonstrate skills across 6 marketing roles.

## Key Findings
- **17,000 customers** and **306,534 transactions** analyzed via SQL Server
- **Influencer** is the top-performing channel (ROAS 2.54) — every $1 returns $2.54
- **Social Media** is losing money (ROAS 0.86) despite receiving 45% of ad budget
- Budget reallocation projected to lift overall ROAS from **1.38 → 1.75** within 90 days

## Project Components
| Component | Deliverable |
|---|---|
| Consumer Research | 3 data-backed personas + 2-page Consumer Insight Report |
| Competitive Intelligence | 4-competitor SQL analysis + positioning matrix |
| Digital Channel Audit | ROAS/CPA/CTR analysis + last-touch attribution model |
| 90-Day Strategy Brief | Positioning statement + campaign calendar + 10 KPIs |
| Executive Dashboard | 4-page Power BI dashboard connected to SQL Server |

## SQL Queries Written
- **Week 1** — Schema setup & data validation (17K customers, 306K transactions)
- **Week 2** — 10 consumer research queries (demographics, NPS proxy, behavioral)
- **Week 4** — 4 competitive intelligence queries (pricing, review scores, social footprint)
- **Week 5** — 5 channel audit queries (CTR, ROAS, CPA, last-touch attribution, master comparison)

## Tech Stack
- **Database**: SQL Server Express + SSMS (`localhost\SQLEXPRESS`)
- **Dashboard**: Power BI Desktop (4 pages)
- **Documents**: Node.js + docx library
- **Design**: Canva (personas + positioning matrix)
- **Language**: SQL (primary), JavaScript (document generation)
